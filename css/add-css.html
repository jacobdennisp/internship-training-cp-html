To Bring CSS inside a HTML Page we have 3 Methods

  1. External CSS 
                        <link href="<path-to-the-stylesheet>" rel="stylesheet" >
  2. Internal CSS 
                    <head>
                    <style> 
                        .someClass{
                            color:black;
                        }
                    </style>
                </head>
  3. Inline Css 
    <div style="color:black"> This is Inline</div>
